# PyTorch 需根据服务器的 CUDA Driver 版本单独安装，不能统一锁定。
# 安装步骤：
#   步骤1) 运行 nvidia-smi 查看 CUDA Version
#   步骤2) 根据下表选择对应命令安装 torch + torchvision：
#
#   ┌──────────────────────┬──────────────────────────────────────────────────────────────────────┐
#   │ nvidia-smi CUDA      │ pip install 命令                                                      │
#   ├──────────────────────┼──────────────────────────────────────────────────────────────────────┤
#   │ CUDA 12.6            │ pip install torch==2.5.1 torchvision==0.20.1 --index-url .../cu124    │
#   │ CUDA 12.4 / 12.5     │ pip install torch==2.5.1 torchvision==0.20.1 --index-url .../cu124    │
#   │ CUDA 12.2 / 12.3     │ pip install torch==2.3.1 torchvision==0.18.1 --index-url .../cu121    │
#   │ CUDA 12.1            │ pip install torch==2.3.1 torchvision==0.18.1 --index-url .../cu121    │
#   │ CUDA 11.8            │ pip install torch==2.3.1 torchvision==0.18.1 --index-url .../cu118    │
#   │ 无 GPU / 驱动过旧     │ pip install torch==2.5.1 torchvision==0.20.1 --index-url .../cpu      │
#   └──────────────────────┴──────────────────────────────────────────────────────────────────────┘
